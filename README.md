# 🪙 Coin Toss Game

A simple and interactive **Coin Toss web application** that simulates flipping a coin and displays whether it lands on **Heads** or **Tails**.

---

## 🚀 Features

- 🎲 Random coin toss simulation  
- ⚡ Instant result (Heads / Tails)  
- 🎨 Clean and simple UI  
- 📱 Responsive design (works on desktop & mobile)  

---

## 🛠️ Technologies Used

- **HTML** – Structure of the webpage  
- **CSS** – Styling and layout  
- **JavaScript** – Logic for random coin toss  

---

## 🎮 How It Works

1. Click the **Toss Coin** button  
2. The coin flips (randomly generated result)  
3. You get either:
   - 🪙 Heads  
   - 🪙 Tails
